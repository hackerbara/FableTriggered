# Claude Interactive Xterm Harness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` for task-by-task implementation, or `superpowers:executing-plans` for inline execution with the same checkpoints. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Node/TypeScript + Playwright + xterm.js harness that drives real interactive Claude Code through a PTY with keyboard, mouse, screen-state, and byte-log assertions.

**Architecture:** A Playwright-controlled browser hosts xterm.js. xterm connects over a byte-safe WebSocket bridge to a loopback-only `node-pty` backend that launches either deterministic fixtures or an explicitly gated Claude binary. Tests assert xterm screen state, terminal modes, sequenced PTY input/output logs, and fixture/Claude-visible effects.

**Tech Stack:** Node 26, TypeScript 6, Playwright, xterm.js 6, node-pty 1.1, ws, Vite, tsx.

## Source spec

Implement against: `docs/superpowers/specs/2026-07-02-claude-interactive-xterm-harness-design.md`

The spec was adversarially reviewed. Preserve its four hard constraints throughout implementation:

1. PTY/WebSocket transport is byte-preserving.
2. The local PTY server is a loopback-only shell boundary with per-run token and origin validation.
3. Real-user mouse tests prove mode precondition, transport bytes, and application effect.
4. Real Claude tests are opt-in and honest about profile/session/network mutation risk.

```ts
export function bytesToBase64(bytes: Uint8Array): string {
  return Buffer.from(bytes).toString('base64');
}

export function base64ToBytes(value: string): Uint8Array {
  return Uint8Array.from(Buffer.from(value, 'base64'));
}

export function binaryStringToBytes(value: string): Uint8Array {
  const out = new Uint8Array(value.length);
  for (let i = 0; i < value.length; i += 1) {
    out[i] = value.charCodeAt(i) & 0xff;
  }
  return out;
}

export function bytesToHex(bytes: Uint8Array): string {
  return Array.from(bytes, byte => byte.toString(16).padStart(2, '0')).join(' ');
}

export function bytesToUtf8(bytes: Uint8Array): string {
  return Buffer.from(bytes).toString('utf8');
}
```

```ts
import type { MouseReportRequest } from './types.js';

function baseButtonCode(button: MouseReportRequest['button']): number {
  switch (button) {
    case 'left': return 0;
    case 'middle': return 1;
    case 'right': return 2;
    case 'wheelUp': return 64;
    case 'wheelDown': return 65;
  }
}

function sgrCode(request: MouseReportRequest): number {
  const base = baseButtonCode(request.button);
  if (request.action === 'drag' && request.button !== 'wheelUp' && request.button !== 'wheelDown') {
    return base + 32;
  }
  return base;
}

export function encodeSgrMouse(request: MouseReportRequest): string {
  const suffix = request.action === 'release' ? 'm' : 'M';
  return `\x1b[<${sgrCode(request)};${request.col + 1};${request.row + 1}${suffix}`;
}

export function sgrMousePattern(request: MouseReportRequest): RegExp {
  const escaped = encodeSgrMouse(request)
    .replace(/[.*+?^${}()|[\]\\]/g, '\\$&')
    .replace('\\x1b', '\\u001b');
  return new RegExp(escaped);
}
```

```ts
import { mkdirSync, writeFileSync } from 'node:fs';
import { join } from 'node:path';
import { bytesToBase64, bytesToHex, bytesToUtf8 } from '../shared/bytes.js';
import type { SequencedBytes } from '../shared/types.js';

export class ArtifactLog {
  private seq = 0;
  readonly input: SequencedBytes[] = [];
  readonly output: SequencedBytes[] = [];

  record(direction: SequencedBytes['direction'], bytes: Uint8Array): SequencedBytes {
    const entry: SequencedBytes = {
      seq: this.seq,
      timestampMs: Date.now(),
      direction,
      bytes: Uint8Array.from(bytes)
    };
    this.seq += 1;
    if (direction === 'pty-input') this.input.push(entry);
    else this.output.push(entry);
    return entry;
  }

  writeTo(dir: string): void {
    mkdirSync(dir, { recursive: true });
    writeFileSync(join(dir, 'pty-input.jsonl'), this.input.map(entryToJson).join('\n') + (this.input.length ? '\n' : ''));
    writeFileSync(join(dir, 'pty-output.jsonl'), this.output.map(entryToJson).join('\n') + (this.output.length ? '\n' : ''));
  }
}

function entryToJson(entry: SequencedBytes): string {
  return JSON.stringify({
    seq: entry.seq,
    timestampMs: entry.timestampMs,
    direction: entry.direction,
    base64: bytesToBase64(entry.bytes),
    hex: bytesToHex(entry.bytes),
    utf8Preview: bytesToUtf8(entry.bytes).replace(/[\u0000-\u001f\u007f]/g, '·').slice(0, 200)
  });
}
```

```ts
import { randomBytes } from 'node:crypto';
import { Buffer } from 'node:buffer';
import { WebSocketServer, type RawData, type WebSocket } from 'ws';
import * as pty from 'node-pty';
import { ArtifactLog } from './artifact-log.js';
import { base64ToBytes } from '../shared/bytes.js';
import type { HarnessLaunchConfig, SequencedBytes } from '../shared/types.js';

interface StartOptions {
  artifactDir: string;
  allowedOrigin: string;
  writeRawLogs?: boolean;
}

interface BrowserInputMessage {
  type: 'input';
  base64: string;
}

interface BrowserResizeMessage {
  type: 'resize';
  cols: number;
  rows: number;
}

type BrowserMessage = BrowserInputMessage | BrowserResizeMessage;

export interface PtyRun {
  token: string;
  wsUrl: string;
  port: number;
  inputLog(): SequencedBytes[];
  outputLog(): SequencedBytes[];
  closed(): boolean;
  close(): Promise<void>;
}

const MAX_INPUT_BYTES = 1_000_000;
const MAX_COLS = 500;
const MAX_ROWS = 200;

export async function startPtyRun(config: HarnessLaunchConfig, options: StartOptions): Promise<PtyRun> {
  const token = randomBytes(24).toString('base64url');
  const log = new ArtifactLog();
  const clients = new Set<WebSocket>();
  const server = new WebSocketServer({ host: '127.0.0.1', port: 0 });
  let closed = false;
  let closePromise: Promise<void> | null = null;

  const child = pty.spawn(config.command, config.args, {
    name: config.termName,
    cols: config.cols,
    rows: config.rows,
    cwd: config.cwd,
    env: { ...process.env, ...config.env, TERM: config.termName },
    encoding: null
  });

  const childExit = new Promise<void>(resolve => child.onExit(() => resolve()));

  function close(): Promise<void> {
    closePromise ??= (async () => {
      if (closed) return;
      closed = true;
      for (const client of clients) client.close();
      clients.clear();
      try {
        child.kill();
      } catch {
        // Already exited.
      }
      await Promise.race([childExit, new Promise<void>(resolve => setTimeout(resolve, 500))]);
      await new Promise<void>(resolve => server.close(() => resolve()));
      if (options.writeRawLogs !== false) log.writeTo(options.artifactDir);
    })();
    return closePromise;
  }

  child.onData((data: string | Buffer) => {
    const bytes = Buffer.isBuffer(data) ? Uint8Array.from(data) : Uint8Array.from(Buffer.from(data, 'utf8'));
    log.record('pty-output', bytes);
    for (const client of clients) {
      if (client.readyState === client.OPEN) client.send(Buffer.from(bytes));
    }
  });

  server.on('connection', (socket, request) => {
    const origin = request.headers.origin ?? '';
    const url = new URL(request.url ?? '/', 'http://127.0.0.1');
    if (origin !== options.allowedOrigin || url.pathname !== '/pty' || url.searchParams.get('token') !== token || clients.size > 0 || closed) {
      socket.close(1008, 'unauthorized');
      return;
    }
    clients.add(socket);
    for (const entry of log.output) socket.send(Buffer.from(entry.bytes));
    socket.on('message', (data, isBinary) => {
      if (isBinary) return;
      const text = rawDataToText(data);
      const message = parseBrowserMessage(text);
      if (!message) return;
      if (message.type === 'input') {
        const bytes = base64ToBytes(message.base64);
        if (bytes.length > MAX_INPUT_BYTES) return;
        log.record('pty-input', bytes);
        child.write(Buffer.from(bytes));
      } else if (message.type === 'resize') {
        if (!Number.isInteger(message.cols) || !Number.isInteger(message.rows)) return;
        if (message.cols < 1 || message.rows < 1 || message.cols > MAX_COLS || message.rows > MAX_ROWS) return;
        child.resize(message.cols, message.rows);
      }
    });
    socket.on('close', () => {
      clients.delete(socket);
      void close();
    });
  });

  await new Promise<void>(resolve => server.once('listening', () => resolve()));
  const address = server.address();
  if (typeof address === 'string' || address === null) throw new Error('expected TCP WebSocket server address');
  const port = address.port;

  return {
    token,
    port,
    wsUrl: `ws://127.0.0.1:${port}/pty?token=${token}`,
    inputLog: () => log.input,
    outputLog: () => log.output,
    closed: () => closed,
    close
  };
}

function rawDataToText(data: RawData): string {
  if (typeof data === 'string') return data;
  if (Buffer.isBuffer(data)) return data.toString('utf8');
  if (Array.isArray(data)) return Buffer.concat(data).toString('utf8');
  return Buffer.from(data).toString('utf8');
}

function parseBrowserMessage(data: string): BrowserMessage | null {
  try {
    const parsed = JSON.parse(data) as Partial<BrowserMessage>;
    if (parsed.type === 'input' && typeof parsed.base64 === 'string') return parsed as BrowserInputMessage;
    if (parsed.type === 'resize' && typeof parsed.cols === 'number' && typeof parsed.rows === 'number') return parsed as BrowserResizeMessage;
    return null;
  } catch {
    return null;
  }
}
```

```ts
    height: rect.height / term.rows
  };
}
```

  else if (mode === 'sgr-any') process.stdout.write('\x1b[?1003h\x1b[?1006h');
  else if (mode === 'legacy-vt200') process.stdout.write('\x1b[?1000h');
  else process.stdout.write('\x1b[?1000h\x1b[?1006h');

  process.stdout.write('\x1b[?1000l\x1b[?1002l\x1b[?1003l\x1b[?1006l');

function render(): void {
  process.stdout.write('\x1b[2J\x1b[H');
  process.stdout.write('Mouse Menu\r\n');
  process.stdout.write(`mode=${mode}\r\n`);
  process.stdout.write('[ click this row ]\r\n');
  process.stdout.write('[ drag target  ]\r\n');

function recordMouse(code: number, col: number, row: number, release: boolean): void {
  if (code === 64) status = `wheel=up row=${row} col=${col}`;
  else if (code === 65) status = `wheel=down row=${row} col=${col}`;
  else if (code >= 32 && !release) status = `drag row=${row} col=${col}`;
  else if (!release) status = `clicked row=${row} col=${col}`;
  else status = `released row=${row} col=${col}`;
}

function handleSgr(value: string): boolean {
  let matched = false;
  const regex = /\x1b\[<(\d+);(\d+);(\d+)([Mm])/g;
  for (const match of value.matchAll(regex)) {
    matched = true;
    recordMouse(Number(match[1]), Number(match[2]) - 1, Number(match[3]) - 1, match[4] === 'm');
  }
  if (matched) render();
  return matched;
}

function handleLegacy(value: string): boolean {
  let matched = false;
  const regex = /\x1b\[M([\s\S])([\s\S])([\s\S])/g;
  for (const match of value.matchAll(regex)) {
    matched = true;
    const code = match[1].charCodeAt(0) - 32;
    const col = match[2].charCodeAt(0) - 33;
    const row = match[3].charCodeAt(0) - 33;
    recordMouse(code, col, row, false);
  }
  if (matched) render();
  return matched;
}

  const value = chunk.toString('binary');
  if (value === '\u0003' || value === 'q') {
    disableMouse();
    process.exit(0);
  }
  handleSgr(value) || handleLegacy(value);
});

```ts
import { spawnSync } from 'node:child_process';
import type { ArtifactRetention, ProfileMode, TargetKind } from '../shared/types.js';

interface ClassifyInput {
  env: Record<string, string | undefined>;
  interactive: boolean;
  copiedPatched: boolean;
}

interface ClassifiedConfig {
  command: string;
  targetKind: TargetKind;
  profileMode: ProfileMode;
  artifactRetention: ArtifactRetention;
  rawArtifactsAllowed: boolean;
  homeOverride?: string;
  warning: string;
}

type ClassifyResult = { ok: true; config: ClassifiedConfig } | { ok: false; reason: string };

export interface SmokeCommandResult {
  exitCode: number | null;
  stdout: string;
  stderr: string;
}

export interface ReadOnlySmokeResult {
  version: SmokeCommandResult;
  help: SmokeCommandResult;
}

export function classifyRealClaudeTarget(input: ClassifyInput): ClassifyResult {
  const command = input.env.CLAUDE_HARNESS_CLAUDE;
  if (input.env.CLAUDE_HARNESS_REAL !== '1') {
    return { ok: false, reason: 'Set CLAUDE_HARNESS_REAL=1 to enable real Claude tests.' };
  }
  if (!command) {
    return { ok: false, reason: 'Set CLAUDE_HARNESS_CLAUDE to the target Claude executable.' };
  }
  const rawArtifactsAllowed = input.env.CLAUDE_HARNESS_REAL_ARTIFACTS_OK === '1';
  if (input.copiedPatched) {
    const homeOverride = input.env.CLAUDE_HARNESS_HOME;
    if (!homeOverride) {
      return { ok: false, reason: 'Copied patched Claude tests must set CLAUDE_HARNESS_HOME to an explicit isolated home/config directory before claiming profile isolation.' };
    }
    return {
      ok: true,
      config: {
        command,
        targetKind: 'copiedPatchedClaude',
        profileMode: 'isolatedHome',
        artifactRetention: 'realPrivateOptIn',
        rawArtifactsAllowed,
        homeOverride,
        warning: 'Copied patched Claude target with explicit isolated home. Do not mutate the live install.'
      }
    };
  }
  if (input.interactive && input.env.CLAUDE_HARNESS_LIVE_PROFILE_MUTATION_OK !== '1') {
    return { ok: false, reason: 'Interactive live-profile Claude tests may mutate ~/.claude, auth/session state, network state, and cwd. Set CLAUDE_HARNESS_LIVE_PROFILE_MUTATION_OK=1 to acknowledge.' };
  }
  return {
    ok: true,
    config: {
      command,
      targetKind: input.interactive ? 'realClaudeInteractiveLiveProfile' : 'realClaudeReadOnly',
      profileMode: input.interactive ? 'liveHome' : 'isolatedHome',
      artifactRetention: 'realPrivateOptIn',
      rawArtifactsAllowed,
      warning: input.interactive ? 'Live-profile mutation run.' : 'Read-only command shape only: --version and --help.'
    }
  };
}

export function smokeReadOnlyClaude(command: string): ReadOnlySmokeResult {
  return {
    version: run(command, ['--version']),
    help: run(command, ['--help'])
  };
}

function run(command: string, args: string[]): SmokeCommandResult {
  const result = spawnSync(command, args, { encoding: 'utf8', timeout: 10_000 });
  return {
    exitCode: result.status,
    stdout: result.stdout ?? '',
    stderr: result.stderr ?? ''
  };
}
```

````markdown
# Claude interactive terminal harness

This harness tests interactive terminal behavior with Playwright, xterm.js, a loopback WebSocket bridge, and node-pty.

## Default test command

```bash
npm install
npx playwright install chromium
npm run test:fixtures
npm run typecheck
```

Default tests run deterministic fixtures only. They do not start real Claude.

## Real-user vs protocol injection

Real-user helpers such as `press`, `type`, `clickCell`, `dragCells`, and `wheel` drive browser events through Playwright. They prove browser -> xterm -> PTY behavior.

Protocol helpers such as `sendRaw` or SGR encoders inject bytes directly. They are useful for lower-level checks but do not prove a browser user action.

## Mouse proof invariant

Every real-user mouse test must prove:

1. xterm mode precondition;
2. transport bytes in the backend input log;
3. visible fixture or Claude effect.

## xterm option defaults

The harness pins these options because macOS Option/Alt behavior can otherwise change input semantics:

- `macOptionIsMeta: true`
- `macOptionClickForcesSelection: false`
- `altClickMovesCursor: false`

Cell coordinates are 0-based. `clickCell(3, 2)` clicks row 3, column 2.

## Real Claude gates

Real Claude tests are private local smoke tests. Raw real-Claude artifacts are disabled unless `CLAUDE_HARNESS_REAL_ARTIFACTS_OK=1` is set; even then, treat them as private.

Read-only command shapes require:

```bash
CLAUDE_HARNESS_REAL=1 CLAUDE_HARNESS_CLAUDE=/path/to/claude npm run test:real
```

Interactive live-profile runs require the second mutation gate:

```bash
CLAUDE_HARNESS_REAL=1 \
CLAUDE_HARNESS_CLAUDE=/path/to/claude \
CLAUDE_HARNESS_LIVE_PROFILE_MUTATION_OK=1 \
npm run test:real
```

Interactive live-profile runs may mutate `~/.claude`, auth/session state, network state, and cwd. Patch-correctness tests should target copied patched binaries, not the live install, and must set `CLAUDE_HARNESS_HOME` before claiming profile isolation.

## Artifacts

Fixture artifacts are retained under ignored development paths. Real-Claude artifacts can contain private content and must not be copied into public handoff material without explicit review.
````
